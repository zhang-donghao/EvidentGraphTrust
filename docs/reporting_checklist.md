# 论文实验撰写检查表

为确保实验部分符合经典机器学习/深度学习论文规范，请在撰写时逐项核对。

## A. 数据集与预处理
- [ ] 明确数据来源及许可，给出下载链接。
- [ ] 描述样本数量、攻击类型分布、时间跨度等统计信息。
- [ ] 说明预处理流程：清洗、标准化、构图策略、窗口参数。
- [ ] 提供训练/验证/测试划分方式及比例，说明是否跨设备/时间划分。
- [ ] 讨论潜在的数据偏差与局限。

## B. 模型与训练
- [ ] 给出模型架构细节：层数、隐藏维度、激活函数、证据层设置。
- [ ] 列出损失函数及其权重，解释证据正则意义。
- [ ] 描述优化器、学习率调度、正则化、训练轮次、早停策略。
- [ ] 说明硬件环境、运行时间、显存占用。
- [ ] 开源代码与配置文件，确保可复现。

## C. 对比实验
- [ ] 选择与任务相关的基线模型，说明调参策略。
- [ ] 使用相同的数据与特征，保证公平比较。
- [ ] 提供多次重复实验的平均值 ± 标准差或置信区间。
- [ ] 对显著性检验做说明（如 t 检验、Wilcoxon）。

## D. 消融与可视化
- [ ] 确认 ``runs/<dataset>/`` 目录下保存了 reliability.png、history.json 等原始可视化文件，确保图表可复现。
- [ ] 分别移除证据正则、先验嵌入、图结构等组件，并报告指标变化。
- [ ] 绘制可靠性图、置信度直方图、校准曲线。
- [ ] 展示典型案例（如错误预测的车辆/设备）并分析原因。
- [ ] 如有超参数敏感性分析，提供相应图表。

## E. 指标与讨论
- [ ] 报告 Accuracy、Precision、Recall、F1、ROC-AUC、PR-AUC。
- [ ] 报告 ECE、Brier Score、NLL 等校准指标。
- [ ] 讨论性能提升的原因与局限，强调不确定性估计的重要性。
- [ ] 对于实际部署，讨论计算开销、在线更新策略。

## F. 附录与复现
- [ ] 在附录列出完整超参数表、模型大小、运行时间。
- [ ] 提供数据处理脚本、配置文件、模型权重下载方式。
- [ ] 如果使用第三方资源（如 W&B），说明追踪 ID/URL。
